# Team Task Manager

A full-stack web app where users can create projects, add tasks, track progress, and manage tasks with role-based access.

## Features

- Signup and Login
- Dashboard
- Create project and task
- Task status tracking
- Mark task as completed
- Delete task
- Admin/Member role access
- Logout

## Tech Stack

Frontend: React + Vite  
Backend: Node.js + Express  
API: REST API  
Deployment: Railway  

## Run Locally

### Backend

```bash
cd backend
npm install
node index.js

### Frontend

cd frontend
npm install
npm run dev
